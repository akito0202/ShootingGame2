ゲーム概要
・プレイヤーは弾を発射して敵を倒します。

・敵をすべて倒すとステージクリア。

・敵や障害物に当たるとゲームオーバー。

・Enterキーを押すことで次のステージに進むか、再スタートができます。


GameFrame: ゲームのメインループと描画・衝突処理を管理。

GameWorld: ゲームの状態（プレイヤー、敵、スコアなど）を保持する静的クラス。

Player: プレイヤーキャラクター。弾の発射や移動を処理。

Enemy, EnemyBase, DropEnemy, CurveEnemy, straightEnemy: 敵キャラクター。+SilentEnemy(途中まで姿が見えない強敵）
PlayerBullet: プレイヤーの弾。

キー	動作
← / →	左右に移動
↑ / ↓	上下に移動
スペースキー	弾を発射
Enter	ステージ移行 or 再スタート

💣 ゲームの流れ
1.	プレイヤー初期化（位置：100, 300）
2.	敵出現（ステージに応じた難易度）
3.	プレイヤーと敵の移動処理
4.	衝突判定（弾・プレイヤー・敵）
5.	敵をすべて倒すと「クリア！」、プレイヤーがやられると「ゲームオーバー」
6.	Enterキーで次ステージ or リトライ
